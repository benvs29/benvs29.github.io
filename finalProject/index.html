Final Project

Here is my final project for Advanced Styling with Responsive Design. Thank you Professor van Lent for a very interesting course. I learned a lot about responsive design. Please enjoy the cute pictures of kittens!

This responsive web page adjusts to the screen size, and also to user preferences. The layout will change when the width reaches 772px, and again when the width reaches 992px. The page also responds to user preferences for color scheme (light/dark) and to reduce motion.
